# Electricity Usage Analysis

## Project Description
This project analyzes electricity consumption data and presents insights through an interactive Power BI dashboard, helping identify usage patterns and optimize energy consumption.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
- Excel / CSV Dataset

## Dataset
The dataset contains household electricity consumption data including:
- Voltage
- Global Active Power
- Global Reactive Power
- Global Intensity
- Sub Metering values
- 
## Key Insights
- Electricity consumption peaks during evening hours
- Usage patterns differ significantly across days
- Certain time periods consistently show high consumption
- Dashboard helps in identifying energy-saving opportunities

## Business Value
- Helps households understand energy usage behavior
- Identifies opportunities to reduce electricity costs
- Enables data-driven decision making for energy efficiency

## Future Improvements
